To call your APIs inspect this page and use the "api_connection" object from the console.

First call api_connection.initialize with your API's URL as a String argument. This will configure the api_connection object to call the API of your choice.

To make API calls: api_connection.[request_type]([url_path_as_array_of_strings], {body_data_as_object})


notes-only API:
: Welcome page -> .get([""])
: Reset DB -> .post(["reset"])
: View all notes -> .get(["all"])
: Create note -> .get(["add"])
: Create note -> .post(["add"], {note: String})
: Read note -> .get([":id"])
: Update note -> .get([":id", "update"])
: Update note -> .post([":id", "update"], {note: String})
: Delete note -> .get([":id", "delete"])
: Delete note -> .post([":id", "delete"])


calc-service-stateless-muber API:
: Do math -> .get([":operation", ":a", ":b"])
  Operations: "add", "multiply", "divide", "subtract"
  "a" & "b" are both required
: Get docs -> .get(["/docs"])


calc-service-stateful API:
: Do math -> .post([":operation"], {a: Number, b: Number})
  Operations: "add", "multiply", "divide", "subtract"
  If no "b" is provided, the last result will be used


notes-and-calc API:
Calc: Do math -> .post(["calculator", ":operation"], {a: Number, b: Number})
  Operations: "add", "multiply", "divide", "subtract"
  If no "b" is provided, the last result will be used
Notes: Welcome page -> .get(["notes", ""])
Notes: Reset DB -> .post(["notes", "reset"])
Notes: View all notes -> .get(["notes", "all"])
Notes: Create note -> .get(["notes", "add"])
Notes: Create note -> .post(["notes", "add"], {note: String})
Notes: Read note -> .get(["notes", ":id"])
Notes: Update note -> .get(["notes", ":id", "update"])
Notes: Update note -> .post(["notes", ":id", "update"], {note: String})
Notes: Delete note -> .get(["notes", ":id", "delete"])
Notes: Delete note -> .post(["notes", ":id", "delete"])
